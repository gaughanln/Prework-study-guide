Prework Study Guide

HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • When possible, use semantic HTML elements as it makes your HTML easier to read by others.
  • Elements nestled inside other elemts are called child elements.
  • The head contains elements that aren't visible to the viewer.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element
  • The selector defines the elements or attributes to which the rules or declartion will apply.
  • There are 3 kinds of CSS style: Inline CSS, Internal CSS Style Sheet, and External CSS Style Sheet. External is most commonly used.

Git

  • git status: shows what branch you're in and if any changes are in progress.
  • git checkout -b branch-name: creates a new branch and switches to it.
  • git help: Gives you a list of git commands and more information on them.
  • git add FileName: adds a file.
  • git commit -m "descriptive message": commits your staged content.
